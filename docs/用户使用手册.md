## 一、快速开始

### 方法 1：双击打开（推荐）✅

1. 打开文件夹 `E:\WorkBuddy\stock-backtest\`
2. **双击 `index.html`**
3. 浏览器自动打开，显示茅台（sh600519）的 K 线图
4. 页面加载约 1~2 秒

> ✅ 无需启动任何服务器，零配置即用。

### 方法 2：通过 HTTP 服务器（可选）

如需加载更多股票数据或使用实时行情功能，可启动本地服务器：

```bash
# 方法 A：使用提供的批处理文件
双击 start-server.bat

# 方法 B：手动启动
cd E:\WorkBuddy\stock-backtest
python -m http.server 8888
```

然后浏览器访问：`http://localhost:8888`

> ⚠️ 实时行情功能在 `file://` 协议下无法使用（浏览器安全限制），需通过 HTTP 服务器访问。

## 三、功能使用指南

### 3.1 查看 K 线图

**默认显示**：茅台（sh600519）日 K 线，最近 800 条。

**操作**：
- 🖱️ 鼠标滚轮：缩放 K 线图
- 🖱️ 鼠标拖拽：左右平移 K 线图
- 📱 手机触屏：双指缩放，单指拖拽

### 3.2 切换技术指标

点击指标按钮栏中的按钮：

| 按钮 | 功能 | 说明 |
|------|------|------|
| MA5 | 5 日均线 | 橙色线，叠加在主图上 |
| MA10 | 10 日均线 | 紫色线，叠加在主图上 |
| MA20 | 20 日均线 | 蓝色线，叠加在主图上 |
| MACD | MACD 指标 | 独立副图，包含直方图 + DIF/DEA 线 |
| RSI | RSI 指标 | 独立副图，14 周期 |
| BOLL | 布林带 | 上轨/中轨/下轨三条线叠加在主图上 |
| 对数 | 切换线性/对数坐标 | 对数坐标适合长期趋势观察 |
| 基本面 | 展开基本面面板 | PE/PB/ROE/股息率 |

> 💡 再次点击按钮可关闭该指标。

### 3.3 搜索/切换股票

1. 在顶部状态栏的搜索框中输入股票代码或名称
2. 系统自动筛选匹配的股票
3. 点击搜索结果中的股票，K 线图自动切换

> ⚠️ 当前仅茅台（sh600519）有内嵌真实数据。切换其他股票将显示模拟数据。  
> 如需添加更多真实数据，请参阅《开发指南》。

### 3.4 运行回测

1. 点击左侧导航栏的 **📈 回测** 图标
2. 在左侧参数面板中配置：
   - **策略选择**：双均线 / MACD
   - **初始资金**：默认 100,000 元
   - **短周期** / **长周期**：双均线策略参数
   - **手续费率**：默认 0.03%
   - **滑点**：默认 0.1%
3. 点击 **"运行回测"** 按钮
4. 查看回测结果：
   - **中间面板**：绩效指标卡片（总收益率、年化、最大回撤、夏普比率、胜率、交易次数）
   - **右侧面板**：资金曲线图
   - **底部面板**：交易记录表格

### 3.5 查看基本面数据

1. 点击左侧导航栏的 **📋 基本面** 图标
2. 基本面面板展开，显示：
   - **PE(TTM)**：市盈率（滚动 12 个月）
   - **PB**：市净率
   - **ROE**：净资产收益率
   - **股息率**
3. 每个指标显示当前值和历史分位数

## 四、常见问题（FAQ）

### Q1：页面打开后一直显示"正在加载..."怎么办？

**A**：通常是因为 JS 文件未正确加载。请确认：
- `lib/vue.global.prod.js` 存在（约 164KB）
- `lib/lightweight-charts.5.0.6.js` 存在（约 177KB）
- 如缺失，重新从 CDN 下载或运行 `git restore lib/` 恢复

### Q2：K 线图显示的价格是 100 左右（模拟数据），不是真实价格？

**A**：说明未加载到真实数据。请确认：
- `embedded-data.js` 文件存在
- 浏览器控制台（F12）中是否有 `[fetchKline] 使用内嵌数据` 日志
- 如无，按 **Ctrl + F5** 强制刷新页面

### Q3：实时行情显示 `--` 正常吗？

**A**：正常。`file://` 协议下浏览器阻止了实时行情 API 调用。  
**解决方法**：启动 HTTP 服务器（`python -m http.server 8888`），然后访问 `http://localhost:8888`。

### Q4：切换股票后显示的是模拟数据，如何获取真实数据？

**A**：运行以下命令之一：
```bash
# 方法 A：使用 Python 脚本生成内嵌数据
python gen_embedded_data.py

# 方法 B：使用 Node.js 抓取 K 线数据
node fetch-kline.js sh600036 800
```
然后将生成的数据手动添加到 `embedded-data.js` 中。

### Q5：页面加载很慢（超过 5 秒）？

**A**：检查是否误引入了 Tailwind CSS CDN。  
打开 `index.html`，确认**没有**以下行：
```html
<script src="https://cdn.tailwindcss.com"></script>  ❌ 应删除
```
正确状态应只引用本地 JS 文件：
```html
<script src="lib/vue.global.prod.js"></script>  ✅
<script src="lib/lightweight-charts.5.0.6.js"></script>  ✅
```

### Q6：手机上能用吗？

**A**：能。系统已适配移动端：
- 左侧导航栏自动隐藏，改为底部导航栏
- K 线图支持触控缩放/拖拽
- 指标按钮栏支持横向滚动
