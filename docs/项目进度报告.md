## 一、项目概况

基于浏览器的股票量化回测系统，使用 Vue 3 + Lightweight Charts v5.0.6 构建，支持深色专业金融终端风格 UI、技术指标叠加、前端回测引擎，数据通过内嵌 JS 文件加载（支持 `file://` 协议直接打开）。

**核心特性**：
- 🖥️ 深色专业终端风格（类同花顺/东方财富 Pro）
- 📊 真实 K 线数据（茅台 800 条，内嵌方案）
- 📈 技术指标：MA / MACD / RSI / BOLL / KDJ
- 🔄 前端回测引擎（双均线、MACD 策略）
- 📁 基本面面板（PE/PB/ROE/股息率）
- 🚀 零依赖离线运行（Vue + LWC 已本地化）
